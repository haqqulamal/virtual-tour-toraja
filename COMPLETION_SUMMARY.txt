    ℹ️  See MIDDLEWARE_SETUP.md for detailed instructions

2️⃣  CREATE STORAGE LINK
    Run command:
    
    php artisan storage:link
    
    (Allows public access to uploaded files)

3️⃣  RUN MIGRATIONS & SEED
    php artisan migrate
    php artisan db:seed
    
    (Creates tables and sample data)

4️⃣  TEST THE APPLICATION
    php artisan serve
    
    Then visit:
    - Home: http://localhost:8000
    - Collection: http://localhost:8000/collection
    - Admin: http://localhost:8000/admin (requires login)

   → Model methods: getLocalizedTitle(), getLocalizedName()

Get all active scenes:
   Scene::active()->ordered()->get()

Get scene with hotspots:
   Scene::with('hotspots')->findOrFail($id)

Get featured artifacts:
   Artifact::featured()->limit(5)->get()

Get artifacts by category:
   Artifact::byCategory($categoryId)->paginate(12)

Search and filter:
   Artifact::where('title_id', 'like', "%search%")
       ->when($categoryId, fn($q) => $q->byCategory($categoryId))
       ->paginate(12)
