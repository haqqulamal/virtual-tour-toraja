DATABASE SCHEMA
===============

CATEGORIES Table
├─ id (PK)

SCENES Table
├─ id (PK)
├─ title (VARCHAR 255)
├─ description (TEXT)

├─ longitude (DECIMAL 11,8)
├─ is_published (BOOLEAN)
├─ order (INTEGER)
└─ timestamps

HOTSPOTS Table
├─ id (PK)
├─ scene_id (FK → scenes.id CASCADE)
├─ title (VARCHAR 255)
├─ description (TEXT)
├─ pitch (DECIMAL 8,4) [-90 to 90]
├─ yaw (DECIMAL 8,4) [0 to 360]
├─ type (ENUM: info, scene, artifact)
├─ linked_scene_id (FK → scenes.id NULLABLE)
├─ artifact_id (FK → artifacts.id NULLABLE)
└─ timestamps

ARTIFACTS Table
├─ id (PK)

├─ name (VARCHAR 255)
├─ description (TEXT)
├─ cultural_significance (TEXT)
├─ image (VARCHAR 255)
├─ keywords (TEXT) [comma-separated]
├─ is_published (BOOLEAN)
└─ timestamps

USERS Table
├─ id (PK)

IMPORTANT NOTES
===============

1. Hotspot Coordinates (Pitch/Yaw):
   - Pitch: -90 (down), 0 (horizontal), +90 (up)
   - Yaw: 0 (north), 90 (east), 180 (south), 270 (west)

2. Panorama Images:
   - Format: Equirectangular projection (360° photos)
   - Size: 2048x1024px minimum recommended
   - Format: JPG or PNG
   - Max: 10MB per file
   - Location: /public/uploads/scenes/

3. Artifact Images:
   - Format: JPG or PNG
   - Max: 5MB per file
   - Location: /public/uploads/artifacts/

4. Language System:
   - Default: Bahasa Indonesia (id)
   - Supported: English (en)
   - Stored in: resources/lang/{locale}/messages.php
   - Session-based: Stored in $_SESSION['locale']

5. Admin Account:
   - Email: admin@toraja.test
   - Password: password
   - Role: admin (required for admin panel access)

6. File Structure:
   - DO NOT move/rename core Laravel directories
   - Upload files go to: public/uploads/{scenes|artifacts}/
   - Database logs: storage/logs/laravel.log
   - Cache: storage/framework/cache/
