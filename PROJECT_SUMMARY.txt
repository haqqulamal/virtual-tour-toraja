## 🚀 NEXT STEPS

1. Run Composer Install
   ```bash
   cd virtual-tour
   composer install
   ```

2. Generate App Key
   ```bash
   php artisan key:generate
   ```

3. Create Database
   - Name: virtual_tour_budaya_toraja
   - Collation: utf8mb4_unicode_ci

4. Run Migrations
   ```bash
   php artisan migrate
   ```

5. Seed Sample Data
   ```bash
   php artisan db:seed
   ```

6. Start Server
   ```bash
   php artisan serve
   ```

7. Access Application
   - Frontend: http://localhost:8000
   - Admin: http://localhost:8000/admin
